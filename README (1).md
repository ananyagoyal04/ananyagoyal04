### 🅿️ Smart Parking Locator System

<img src="https://github-readme-stats.vercel.app/api/pin/?username=ananyagoyal04&repo=ananyagoyal04&theme=transparent&title_color=b8860b&text_color=333333&icon_color=feff7f&border_color=feff7f" alt="Smart Parking Locator System"/>

</div>

Smart Parking Locator System is a web-based application that helps users find and reserve nearby parking spaces in Bangalore using real-time availability and location services, providing a secure, efficient, and user-friendly parking experience.

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript |
| Backend | Node.js, Express.js |
| Database | PostgreSQL |
| Location Services | Maps API |
| Authentication | JWT Authentication |
| Payments | Payment Gateway Integration |

**Features**
- 🔍 Real-time parking space availability
- 📍 Location-based search using Maps API
- 🔐 Secure user authentication with JWT
- 💳 Integrated payment gateway
- 🅿️ Reservation system for nearby parking spaces
