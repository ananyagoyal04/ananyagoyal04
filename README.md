## 🚀 Featured Projects

### 🅿️ Smart Parking Locator System

<img src="https://github-readme-stats.vercel.app/api/pin/?username=ananyagoyal04&repo=ananyagoyal04&theme=nord&hide_border=true&bg_color=ffffff&title_color=feff7f&icon_color=feff7f&text_color=333333" alt="Smart Parking Locator System Pin"/>

Smart Parking Locator System is a web-based application that helps users find and reserve nearby parking spaces in Bangalore using real-time availability and location services. It provides a secure, efficient, and user-friendly parking experience.

**Layer | Technology**

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript |
| Backend | Node.js, Express.js |
| Database | PostgreSQL |
| Location | Maps API |
| Auth & Security | JWT Authentication |
| Payments | Payment Gateway Integration |

**Features**
- 🌊 Real-time parking availability lookup
- 📍 Location-based search via Maps API
- 🔒 Secure JWT-based authentication
- 💳 Integrated payment gateway for reservations
- 🚀 User-friendly reservation flow
